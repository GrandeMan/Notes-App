# Notes-App
A notes app made using React libraries. Notes are automatically saved to a database.

### Features

- `Note Preview`
- `Title Note by First Line`
- `Sort Notes by Last Edited`
- `Delete Notes`
- `Auto-save (w/debouncing)`
- `Customize Sidebar Width`
